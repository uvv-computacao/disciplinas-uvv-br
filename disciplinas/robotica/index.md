# Robótica e Dispositivos Inteligentes

Entenda os fundamentos de eletrônica, computação e microcontroladores para
a construção de robôs e dispositivos inteligentes.
{: .fs-6 .fw-300 }

Esta disciplina é uma introdução à robótica e aos dispositivos inteligentes para
que você possa começar a criar seus primeiros projetos e dispositivos
com microcontroladores. Você aprenderá a utilizar o Arduino como plataforma de
desenvolvimento para microcontroladores AVR de 8 bits. Ao final da disciplina
você construirá um robô seguidor de linha.

{: .attention }
> Para as aulas práticas da disciplina os alunos serão divididos em grupos de 4
> alunos para as tarefas no **Laboratório de Hardware**. O laboratório tem
> diversos materiais, equipamentos e componentes disponíveis para os projetos
> mas, para o projeto final da disciplina, cada grupo deverá adquirir um ``Kit
> de Robô Seguidor de Linha'' (disponíveis na Internet por volta de R$ 70,00; o
> professor indicará os kits adequados).
