# Arquitetura e Organização de Computadores II

Construa projetos com microcontroladores, aprenda arquiteturas de computação
paralela e crie programas para o Cluster de Alta Performance da UVV!
{: .fs-6 .fw-300 }

Esta disciplina é dividida basicamente em uma pequena parte teórica a respeito
de arquiteturas paralelas e um grande projeto prático de construção de um
dispositivo microcontrolado (em geral usamos um microcontrolador AVR de 8 bits,
mas outros microcontroladores também podem ser utilizados).

O projeto prático é escolhido e realizado em grupos de 4 alunos, sob orientação
do professor. Ao final do semestre o cada grupo tem que entregar um dispositivo
funcionando, com uma placa de circuito impresso personalizada, alimentação
independente e outros requisitos.

{: .important }
> Uma boa parte da avaliação do projeto é baseada no **Caderno de Engenharia**
> do grupo. É absolutamente fundamental que cada grupo mantenha seu caderno
> atualizado e preenchido corretamente.

{: .danger }
> Apesar do **Laboratório de Hardware** tem muitos equipamentos, componentes e
> dispositivos à disposição dos alunos, como cada grupo escolhe um projeto de
> interesse qualquer, com certeza cada grupo precisará comprar materiais,
> componentes e dispositivos particulares para seu projeto. Os projetos prévios
> mostram que os grupos gastam, em média, de R$ 200,00 a R$ 500,00 com material
> particular para o projeto (por curiosidade, o grupo que mais gastou em um
> projeto foi um grupo que fez um drone controlado por rádio-freqüência do
> zero, cujos materiais custaram aproximadamente R$ 1.400,00). Assim, cada grupo
> deve se preparar para adquirir os materiais necessários que não estão
> disponíveis no Lab. de Hardware. Por último, para aprender a projetar uma PCB
> de forma correta usando o software KiCad, cada grupo também precisará adquirir
> um curso na Udemy.

{: .danger }
> É **IMPOSSÍVEL** concluir o projeto prático se você e seu grupo trabalharem no
> Laboratório de Hardware apenas durante a carga horária de aula normal da
> disciplina. Reserve sua agenda pois você terá que trabalhar aos **SÁBADOS**,
> no período de 08:00h até 12:00h, durante várias semanas ao longo do
> semestre. Além do trabalho aos sábados você também precisará trabalhar durante
> várias tardes durante as semanas, conforme a disponibilidade de horários do
> Laboratório de Hardware.
